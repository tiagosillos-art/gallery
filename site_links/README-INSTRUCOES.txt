Arquivos prontos para teste no site:
- index.html
- style.css
- script.js

Importante:
- O método continua por links externos, sem conversão para imagens locais.
- Para trocar pelas suas imagens, edite apenas o arquivo script.js.
- Procure a constante CATEGORIES.
- Substitua cada URL pelos seus próprios links diretos de imagem.
- Mantenha as aspas e as vírgulas.

Exemplo:
"https://seusite.com/imagens/foto-01.jpg"

Para testar:
- suba os 3 arquivos no mesmo diretório do seu site
- abra o index.html

Observação:
- O Three.js continua sendo carregado por CDN no index.html.
